# planning_app


# install and setup

`pip install pipenv`

clone the repo and cd into the project dir
`cd planning_app`

activate pipenv
`pipenv shell`

install dependancies from pipfile
`pipenv install`

change dir to django project
`cd planning_app`

run django server
`pipenv run python manage.py runserver`

by default server will start on http://127.0.0.1:8000/
